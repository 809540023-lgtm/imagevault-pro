# ImageVault Pro - 正式發布版本 v1.0.0

## 🌟 專案簡介
ImageVault Pro 是一個整合了高效能後端 API 與直覺式前端介面的圖片管理系統。本專案已在 Manus 平台完成正式發布，具備完整的數據持久化能力。

## 🏗️ 系統架構
- **後端服務**: 基於 FastAPI 的非同步架構，支援高併發圖片處理。
- **前端平台**: 包含單一入口 Dashboard 與 HTML5 智能拍照介面。
- **數據存儲**: 使用非同步 SQLite 確保元數據安全，並支援本地/雲端圖片存儲。

## 🔗 正式入口連結
- **管理主控台 (Dashboard)**: `/dashboard`
- **智能拍照介面 (Camera)**: `/static/camera.html`
- **開發者文件 (Swagger)**: `/docs`
